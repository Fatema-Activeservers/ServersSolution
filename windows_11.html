Performance and Efficiency

With Windows 11, Microsoft has optimized the operating system to deliver better performance and efficiency. Windows 11 Minimum System Requirements

It's important to note that these are the minimum system requirements to buy Windows 11.

Processor

1 gigahertz (GHz) or faster with 2 or more cores on a compatible 64-bit processor or System on a Chip (SoC).

RAM

Computer should have at least 4 GB of RAM installed as per the minimum system requirements for Windows 11.

Storage

Microsoft Windows 11 requires a minimum of 64 GB of storage or larger on the device to install and operate efficiently.

System firmware

The system firmware requirement for Microsoft Windows 11 is UEFI (Unified Extensible Firmware Interface) and Secure Boot capable.

Graphics card

The graphics card requirement to buy Windows 11 is that it should be compatible with DirectX 12 or later with a WDDM 2.0 driver.

Display

The display requirement for Windows 11 is a high definition (720p) display that is greater than 9 inches diagonally, with 8 bits per color channel.
