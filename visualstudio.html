What is the difference between Visual Studio and Visual Studio Code?

Visual Studio is an Integrated Development Environment, also known as an IDE. And Microsoft Visual Studio Code, commonly known as VS Code, is a lightweight and powerful source code editor developed by Microsoft. While it shares the Visual Studio name, it is different from the full-fledged Visual Studio IDE and is primarily focused on providing a streamlined and customizable coding experience.
