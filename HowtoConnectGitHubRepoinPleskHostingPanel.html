Plesk Hosting

Plesk is best web hosting control panel for windows and linux os. Plesk provides GUI options to Admins, developers to manager servers settings. With GUI options, It is easy to manage PHP,.NET, Database, Node, FTP, server configuration and many more settings. If you have purchased Plesk license from ServersSolutions then customer will get free support. Plesk provides Git Hub extension to connect with code repo.

Please Follow Below steps to connect Plesk and GitHub

  • Login to GitHub, Upload your code and get your GitHub Repo URL. e.g. https://github.com/sample/Sanoke.git
  • Login To Plesk and Go to Plesk github extension
  • Now Click on Add repository
  • Now select Remote repository and enter your Repository URL (You can find URL from github portal). Click on Create.
  • It will take 2 min to 5 min to create connection between plesk and github
  • Now, Just click on PULL button to fetch your all code changes to plesk.
  • In Above Image, You can check the updated code in published in plesk.
